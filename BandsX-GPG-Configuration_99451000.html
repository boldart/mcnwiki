MCNi Wiki : BandsX GPG Configuration

Created by Stephen Smith on Oct 24, 2018

This is how GPG/PGP (encryption) should be configured on new band site hosting servers.

See also: Creating a Band Site Hosting Server

# yum install gpg gpgme gpgme-devel
# pecl install gnupg
# echo "; Enable GPG module
> extension=gnupg.so" > /etc/php.d/gnupg.ini

# cp /usr/bandsXtools/home_dir/gnupg.tar.gz ~
# cd ~
# tar -xvzf gnupg.tar.gz
