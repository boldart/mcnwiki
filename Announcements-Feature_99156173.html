MCNi Wiki : Announcements Feature

Created by Stephen Smith on Oct 24, 2018

Description

This feature disseminates announcements from bombplates.com to all the hosted sites. Announcements are posted to bombplates.com. Then, once per day, each band site queries bombplates.com for any new announcements and clears old announcements out.
