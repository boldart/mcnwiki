MCNi Wiki : CAPTCHA Feature

Created by Stephen Smith on Oct 24, 2018

Description



Whenever content can be submitted/created by fans, the band may enable a CAPTCHA to prevent spam. Users have a CAPTCHA management page where they can switch between no CAPTCHAS, reCAPTCHA, or 3 levels of Drupal's built-in image captcha.



In the future, we hope to implement Stupid Filter as a secondary layer of spam protection. i.e. users would enable Stupid Filter and set a strictness level. Any posts that tripped the Stupid Filter would either be blocked or put in the moderation queue.
