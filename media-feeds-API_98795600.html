media_feeds module - Hooks created by the media_feeds module
 - These hooks pertain to embedding video players in the site

hook_media_feeds_list
  Parameters:
    none
  Returns:
    array(
      'h_name' => string - human-readable name
      'name' => string - machine-readable name
      'user_allowed' => bool - check user's permission to admin this feed
    )
  Description:
    Describe this module for media_feed parent module

hook_media_feeds_admin
  Parameters:
    none
  Returns:
    array() - per forms API
  Description:
    Return form elements for administering this module from manage/media_feeds

hook_media_feeds_admin_validate
  Parameters:
    $form => array() - per forms API
    &$form_state => array() - per forms API
  Returns:
    none
  Description:
    Form validation for hook_media_feeds_admin.

hook_media_feeds_admin_submit
  Parameters:
    $form => array() - per forms API
    &$form_state => array() - per forms API
  Returns:
    none
  Description:
    Form submission for hook_media_feeds_admin.

hook_media_feeds_block
  Parameters:
    $which_block => int - block number (usually 0)
  Returns:
    string - HTML code
  Description:
    Generate the HTML to display this media feed as a block
