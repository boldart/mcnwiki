MCNi Wiki : Feature List - Core Site

Created by Stephen Smith on Oct 23, 2018

This is a list of features found on bombplates.com

Drupal Core - The drupal core
Third-party features - Modules/features built and maintained by third parties.
Account Commands - Send system commands to the band sites server to manage sites. 

Account Forms - Account managers have access to a series of forms allowing them to set up, finalize, and tear down band sites.
Announcements Server - Announcements can be posted to bombplates.com which will be sent out to all the band sites once per day.

Authorize.net - User payments are handled through Authorize.net. Forms are available to allow them to enter/update their payment info
Bug Report - A form is generated on bombplates.com that allows users to submit detailed bug reports to Bombplates.
Contact Bombplates - A form is generated on bombplates.com that allows users to contact various branches of Bombplates.

Password Integration - Passwords are synched automatically between bandsX.bombplates.com and bombplates.com 

Referral Report - A monthly report of which accounts were referred by which referral partners is available.
Timezones - Determine what timezone an address is in.
User Tables - Generate tables listing all paying/suspended/trial users.
