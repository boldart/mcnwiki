MCNi Wiki : Referral Report Feature

Created by Stephen Smith on Oct 23, 2018

2018 comment: Nope. This feature does not exist as described. The referral program was only ever vestigial at best.

Description

A monthly report is generated on bombplates.com. This allows billing staff to view which paying sites were referred to the system by which referral partners. A daily report is available showing the status of references per day. There is also an end-of-month report generated on the last day of the month.
