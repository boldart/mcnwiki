MCNi Wiki : Site Settings Feature

Created by Stephen Smith on Oct 24, 2018

Description

This feature allows users to configure certain global settings. These settings include - but aren't limited to - meta tags, deactivating the site, and the band name.

These settings can be configured from Settings > Site Settings > General
