MCNi Wiki : Drupal Core Features - Core Site

Created by Stephen Smith on Oct 23, 2018

Description:

 
Bombplates is currently based on Drupal core 6.x. As time progresses, we will (and have historically) customize, replace, and phase out Drupal code and phase in custom code. When official support for version 6.x ends, we will fork entirely.
