MCNi Wiki : Tour Feature

Created by Stephen Smith on Oct 24, 2018

Description

This feature allows bands to post tour dates on their page. They can configure the system to automatically post links to new tour dates on various social networking sites.

Tour date content can include a title, a date, a venue name, a description, a start time, a door time, a ticket price, a link to buy tickets, a venue address (including city, state, postal code, country, phone number, URL, etc.), and additional artists.
