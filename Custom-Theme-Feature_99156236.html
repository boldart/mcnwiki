MCNi Wiki : Custom Theme Feature

Created by Stephen Smith on Oct 24, 2018

Description

This feature allows bands to switch to a fully-customized theme. It gives them a management interface to tweak the theme and generates the HTML/CSS code to match.

There is also a sandbox mode where logged-in users can see the new theme, but visitors cannot.
