MCNi Wiki : Mailing List Feature

Created by Stephen Smith on Oct 24, 2018

Description

This feature allows bands to embed a Widget directing fans to sign up for the band's mailing list. Sub-modules will integrate with specific 3rd-party mailing lists. There is a management interface allowing mailing list selection and configuration.
