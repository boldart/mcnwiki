Created by Stephen Smith on Oct 24, 2018

Testing steps to ensure your server is configured correctly

See also: Creating a Band Site Hosting Server

Launch a test site

# cd /usr/bandsXtools
# /usr/bandsXtools/newSite.pl -v -v -u newserver.com -b "Test Band" -d newserver -e newserver@devnullmail.com
# /usr/bandsXtools/newSite-theme.pl -v -v -url newserver.bombplates.com -t 1111 -b "Test Band" -usr test -e newserver@devnullmail.com -pw password -k 0000
# /usr/bandsXtools/finalizeTheme.pl -v -v -u newserver.bombplates.com

If any of those steps fail (or when you're done testing the server)

# /usr/bandsXtools/delSite.pl -v -v -u newserver.com -s newserver

Create a hosts entry pointing www.newserver.com, newserver.com,  and newserver.bombplates.com to the new server on your local system

log into www.newserver.com with the credentials test/password

Click around a bit to make sure everything works correctly

  • Sync with some external services: a mailing list, twitter, facebook
