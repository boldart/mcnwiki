MCNi Wiki : Merch Feature

Created by Stephen Smith on Oct 24, 2018

Description

This feature allows bands to list and sell merchandise from their site. We DO NOT handle financial transactions. This feature merely allows them to embed HTML to redirect them to 3rd-party payment gateways. Currently, they can embed fully-customized HTML generated by another site as well as creating PayPal and Merchmo buttons from scratch. Support for more stores/carts/payment gateways is fothcoming. There is a designated merch page, and items may be listed on the front page too. There is a management page.

Merch items have a title, type, price, image, details, a weight for sorting, and whatever fields the payment type (custom, PayPal, etc) adds.
