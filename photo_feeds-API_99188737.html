photo_feeds module - Hooks created by the photo_feeds module
 - These hooks pertain to embedding photo slideshows in the site

hook_photo_feeds_list
  Parameters:
    none
  Returns:
    array(
      'h_name' => string - human-readable name
      'name' => string - machine-readable name
      'user_allowed' => bool - check user's permission to admin this feed
    )
  Description:
    Describe this module for the photo_feed parent module

hook_photo_feeds_admin
  Parameters:
    none
  Returns:
    array() - per forms API
  Description:
    Return form elements for administering this module from manage/photo_feed

hook_photo_feeds_admin_validate
  Parameters:
    $form => array() - per forms API
    &$form_state => array() - per forms API
  Returns:
    none
  Description:
    Form validation for hook_photo_feeds_admin.

hook_photo_feeds_admin_submit
  Parameters:
    $form => array() - per forms API
    &$form_state => array() - per forms API
  Returns:
    none
  Description:
    Form submission for hook_photo_feeds_admin.

hook_photo_feeds_content
  Parameters:
    none
  Returns:
    string - HTML code
  Description:
    Generate the HTML to display this photo feed
