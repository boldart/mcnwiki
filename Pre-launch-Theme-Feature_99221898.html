MCNi Wiki : Pre-launch Theme Feature

Created by Stephen Smith on Oct 24, 2018

Description


This feature allows a band to disable its site. A site in pre-launch mode does not show its normal theme or content to to anonymous user. The band can still see and edit the site, but fans only see an "under construction" page.
