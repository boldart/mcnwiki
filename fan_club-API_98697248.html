Created by Stephen Smith on Oct 23, 2018

fan_club hooks - hooks created by the fan_club module
 - These hooks allow other modules to react to changes to fan club status

hook_fan_club_expire_membership
  Parameters:
    $account => object - a user object
  Returns:
    none
  Description:
    Allow modules to react when a user's fan club membership expires.

hook_fan_club_extend_membership
  Parameters:
    $account => object - a user object
  Returns:
    none
  Description:
    Allow modules to react when a user's fan club membership is extended (or created).
