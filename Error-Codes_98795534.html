MCNi Wiki : Error Codes

Created by Stephen Smith on Oct 23, 2018

We may sometimes instruct users to contact us with error codes.

Here's what they mean:
Breakdown of code #s: Codes are 8 digits comprising 4 2 hex digit ID codes.
1 - Server ID (bombplates.com is 01, bands.bombplates is 02, etc.)
2 - Module ID. Make one up arbitrarily. (Unique per module)
3 - File ID. In ascending, alphabetical order whenever possible.
4 - Error ID. Starting from 01 and ascending within the file.
