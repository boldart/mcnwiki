MCNi Wiki : BandsX Facebook App Configuration

Created by Stephen Smith on Oct 24, 2018

Here is how the Facebook app should be configured for new band site servers.

See also: Creating a Band Site Hosting Server

Due to how Facebook performs validation, each instance of bandsX.bombplates.com will need its own instance of the Bombplates Facebook app.

Create a new app. Configure it to match one of the existing apps. (We can't maintain up-to-date specifics of that configuration because it keeps changing)

Add its information the server's facebook configuration file.

open /var/www/include/this_node/facebook.php

$appsecret and $appid are found in the App details on Facebook's side.
