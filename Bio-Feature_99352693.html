MCNi Wiki : Bio Feature

Created by Stephen Smith on Oct 24, 2018

Description

This feature allows bands to post biographies to their page. Each biography includes a name, picture, text description, and display order. There is a private biography management page for the band and a public biography display page that may or may not be linked in the navigation bar.
