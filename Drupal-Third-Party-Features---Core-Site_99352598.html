MCNi Wiki : Drupal Third Party Features - Core Site

Created by Stephen Smith on Oct 23, 2018

This is everything in the modules directory that isn't custom-made for Bombplates


addthis

admin_theme - Admin and public pages have a different theme
advanced_help - This module generates documentation from files within the module folder.
better_formats - Offer some alternate formats for text input
cck - Content construction kit
checkbox_validate - Throws a validation error if the "terms and services" checkbox is unchecked on new signups
colorbox
content_taxonomy - Adds a taxonomy field to CCK
content_taxonomy_image - Adds images to the content_taxonomy field
date - Adds a javascript calendar widget to for date fields
disqus - Integrates with Disqus. WE DON"T ACTUALLY USE THIS! Uninstall it.


emma - Custom MCN module. integrate with the MyEmma API. Not enabled.

emma_block
emma_register - Create a checkbox to sign up new users to the MyEmma mailing list.
filefield - Adds a file field to CCK
formfilter - Does some form alterations. DOES NOT APPEAR TO BE IN USE.
getid3 - Extracts metadata from uploaded media files
globalredirect - Redirects users to more appropriate URLs if they enter the wrong one (e.g. non-clean URLs)
google_analytics - Embeds Google Analytics tracking code on pages.
hierarchical_select - Creates a hierarchy of taxonomy terms.
imageapi - Includes image manipulation libraries for other modules to use.
imagecache - Automatically crop and resize images based on URL.
imagefield - Adds an image field to CCK
jquery_ui - Includes some jquery libraries for other modules. MAY NOT BE IN USE.
jquery_update - Update to a more recent version of the jquery javascript library. MAY NOT BE IN USE. I THINK WE'RE DOING IT MANUALLY.
lightbox2
logintoboggan - Redirect users on login. New users go to whichever form is relevant to their current stage of site setup. Most other users are sent to their own user details page.
mailchimp
pathauto - Automatically create aliases for new content.
permission_select - Allows fine-control of permissions.
profile_category_weight - Re-orders fields on the user registration from.
securepages - Automatically redirects users to https when viewing admin, user, or payment pages.
taxonomy_image - Associates images with taxonomy terms.
taxonomy_redirect - Redirect users elsewhere in the blog when they click on Authors or Tags in the blog.
thickbox - Lightbox pop-ups.
token - Provides a tokenization library for other modules.
transliteration - Sanitizes text to the Latin alphabet
views - Views
views_atom
webform
webform_mailchimp
wysiwyg - Generates a WYSiWYG textarea for blog posts, etc.



xmlsitemap - Generates a sitemaps.org-compatible sitemap. It can be found here.

zendesk - Integrates the site with Zendesk. THIS SHOULD BE REMOVED. We're using uservoice now.
