MCNi Wiki : APIs

Created by Stephen Smith on Oct 23, 2018

Bombplates supports two APIs for extending functionality

  • The external API allows 3rd-party sites to access some features of the site and perform functions available to band members.
  • The internal API allows modules to extend the functionality of the Bombplates system itself quickly and easily while maintaining consistent code internally and a consistent visual appearance externally
